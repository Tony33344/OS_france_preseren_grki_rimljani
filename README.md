# Grčija & Rim — Interaktivni učni pripomoček

Sodobno didaktično gradivo za 9. razred osnovne šole o primerjavi načinov vladanja v antični Grčiji in Rimu.

## Značilnosti

- **Zgodovinska časovnica** — ključni dogodki od prve olimpijade do propada Zahodnega rimskega cesarstva
- **Primerjava oblik vladavine** — vizualna primerjava demokracije, aristokracije, monarhije, tiranske in republikanske oblike
- **Ključne osebnosti** — Periklej, Knez Kleon, Julij Cezar, Ciceron, Avgust
- **Interaktivni kviz** — preverjanje znanja s takojšnjim povratnim odzivom
- **Viri** — akademsko preverljivi viri za nadaljnje raziskovanje

## Tehnologije

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Razvoj

```bash
npm install
npm run dev
```

## Gradnja za produkcijo

```bash
npm run build
```

Izhodna koda se nahaja v mapi `dist/`.

## Avtor

Izdelano za 9. razred osnovne šole.
